Graph Visualization: Visualizing relationships among keywords of research publications

Graded Homework Assignment 03

Lecture Grundlagen der Informationsvisualisierung, University of Bamberg, summer semester 2023

Explanations:

  • I took the liberty of modifying the template to conform to current web development standards, e.g. moving the data and the javascript code to separate files.
  • First, I filtered the given data and determined the nodes (=keywords) and their weight (=number of publications associated with a keyword)
  • After that, I did the same for the edges (=publication sharing both keywords) and their weight (=number of publications sharing both keywords)
  • Then I was able to exchange the dummy data with the actual data, leading to a graph.
  • I slightly adjusted the forces so that the provided space is used optimally.
  • Subsequently, I added a legend showing the node size encoding.
  • The interactivity was obtained by a click event on a node.
  • Clicking on a node sets its opacity to 1. Its name, the number of the associated publications and the list of associated publications are appended to the grey div next to the graph.
  • I then adjusted the graph layout: I filtered the nodes and the links so that only the nodes with a weight >= 3 are shown.
  • Finally, I changed the link weight to link.weight/Math.min(node1.weight, node2.weight) and adjusted the filter so that only the links with a weight >= 0.5 are displayed.
